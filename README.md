# Majlis

A social platform focused on Islamic content, enabling users to share posts, connect with others, and engage in meaningful discussions through a modern and interactive experience.

## Features

* User Authentication (Login & Register)
* Home Feed with Infinite Scrolling
* Create, Edit, and Delete Posts
* AI-Powered Content Moderation
* AI Caption Suggestions
* Post Likes & Interactions
* Comments System
* Real-Time Notifications
* Follow & Unfollow Users
* Friends System
* Real-Time Chat
* User Profiles
* Search Users & Posts
* Islamic Categories (Quran, Hadith, Fiqh)
* User Discovery & Suggestions
* Account Settings
* Responsive Design
* Custom 404 & Error Pages

## Tech Stack

### Frontend

* React.js
* TypeScript
* Tailwind CSS
* React Query
* Socket.IO Client

## Getting Started

### Frontend

```bash
npm install
npm run dev
```
